
# মো: তাসনিম আল-হুজাইফা মিরাজ | পূর্ণাঙ্গ জীবনবৃত্তান্ত (CV Website)

[![Live Demo](https://img.shields.io/badge/Live-Demo-brightgreen?style=for-the-badge)](https://md-tasnem.github.io/MD.TASNEM-AL-HUJAIFA-MIRAZ/)

[![Responsive](https://img.shields.io/badge/Responsive-Yes-orange?style=for-the-badge)](#)

এটি একটি পেশাদার ও আধুনিক **পোর্টফোলিও / জীবনবৃত্তান্ত (CV)** ওয়েবসাইট। এটি মো: তাসনিম আল-হুজাইফা মিরাজ-এর ব্যক্তিগত তথ্য, শিক্ষাজীবন, দক্ষতা, দৈনন্দিন রুটিন, আদর্শ জীবনসঙ্গীর মানদণ্ড এবং যোগাযোগের মাধ্যমগুলো সুন্দরভাবে উপস্থাপন করে।

ওয়েবসাইটটিতে **ডার্ক মোড / লাইট মোড**, টাইপিং অ্যানিমেশন, রেসপনসিভ ডিজাইন এবং প্রফেশনাল কার্ড-ভিত্তিক লেআউট যুক্ত করা হয়েছে।

---

## 🚀 বৈশিষ্ট্যসমূহ (Features)

- 🌗 **ডার্ক মোড ও লাইট মোড টগল** – ব্যবহারকারীর পছন্দ অনুযায়ী থিম পরিবর্তন (লোকাল স্টোরেজে সংরক্ষিত)
- ✍️ **টাইপিং অ্যানিমেশন** – হিরো সেকশনে পেশা ও পরিচয় চমৎকারভাবে পরিবর্তিত হয়
- 📱 **পূর্ণ রেসপনসিভ** – মোবাইল, ট্যাব ও ডেস্কটপে সুন্দরভাবে দেখা যায়
- 🎨 **আধুনিক ডিজাইন** – গ্লাসমরফিজম, গ্রেডিয়েন্ট বর্ডার, হোভার ইফেক্টস
- 📄 **সম্পূর্ণ সিভি** – শিক্ষা, প্রশিক্ষণ, দক্ষতা, ব্যক্তিগত তথ্য সারণী আকারে
- 🕋 **দৈনন্দিন রুটিন** – সময়সূচি ও খাদ্যাভ্যাস লিস্ট করা
- 💍 **আদর্শ জীবনসঙ্গী** – ধর্মীয় ও ব্যক্তিগত প্রত্যাশা স্পষ্টভাবে উল্লেখ
- 🔗 **সোশ্যাল নেটওয়ার্ক** – গিটহাব, ফেসবুক, লিংকডইন, ইউটিউব, ওয়েবসাইট লিংক

## 🔧 কীভাবে ব্যবহার করবেন (How to Use)

### ১. লোকালি রান করা (Run Locally)
```bash
git clone https://github.com/MD-TASNEM/MD.TASNEM-AL-HUJAIFA-MIRAZ.git
cd MD.TASNEM-AL-HUJAIFA-MIRAZ.
# শুধু index.html ব্রাউজারে ওপেন করুন
```

অথবা সরাসরি **Live Server** (VS Code এক্সটেনশন) দিয়ে রান করলে ভালো।

### ২. নিজের তথ্য আপডেট করা (Customize)
- `index.html` ফাইলটি খুলুন।
- আপনার নাম, ঠিকানা, শিক্ষা তথ্য, দক্ষতা, রুটিন ইত্যাদি সম্পাদনা করুন।
- প্রোফাইল ছবি পরিবর্তন করতে `MD_TASNEM.jpg` রিপ্লেস করুন এবং ফাইলের নাম ঠিক রাখুন।
- সোশ্যাল মিডিয়া লিংক আপনার প্রোফাইল অনুযায়ী আপডেট করুন।
- দরকার হলে `footer` এর কপিরাইট ইয়ার পরিবর্তন করুন।

### ৩. গিটহাবে পাবলিশ করা (GitHub Pages)
1. রিপোজিটরি তৈরি করুন (যেমন: `your-username.github.io`)
2. সব ফাইল আপলোড করুন
3. সেটিংস → Pages → Branch: `main` → সেভ করুন
4. কিছুক্ষণের মধ্যে লাইভ হবে।

## 🌟 কাস্টমাইজেশন গাইড (Customization Tips)

### থিম কালার পরিবর্তন
CSS ভেরিয়েবল ব্যবহার করা আছে। `:root` ও `body.dark` সেকশনে গিয়ে **gold** ও **cyan** কালার পরিবর্তন করতে পারেন।

```css
:root {
  --gold: #b56a1e;   /* আপনার পছন্দের সোনালি রং */
  --cyan: #0c4e6e;   /* আপনার পছন্দের সায়ান রং */
}
```

### টাইপিং টেক্সট পরিবর্তন
`<script>` ট্যাগের `texts` অ্যারেটি সম্পাদনা করুন:

```javascript
const texts = ["ওয়েব ডেভেলপার", "আরবি সাহিত্য গবেষক", "আপনার টাইটেল"];
```

### নতুন সেকশন যোগ করা
অন্য সেকশনের মতো `<section class="section">` ব্লক তৈরি করুন এবং নেভিগেশনে লিংক যোগ করুন।
